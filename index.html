a short hike ：一场不期而遇的小小徒步

发表于 2023-01-02 更新于 2023-01-03 分类于 玩游戏

一场不期而遇的小小徒步

无论从细节的设计还是音乐的恰好响起，这个小小的游戏带给我了丰富的感受。海滩上的玩耍、山腰的赛跑，再到从山上一跃而下，身后曳过白色航迹的轻快，每一个细节都营造出了某个在海边嬉戏的暑假般的氛围。每个npc的设计都恰到好处，不是交互的对象而是想塑造真实的人（动物？），他们忙于自己的事情，但也愿意和你聊聊天。没有细致的画质，没有深刻的剧情，但它却轻而易举地让人沉醉其中。

独立游戏不可能做的面面俱到，流程很短，但是韵味深长，a short hike做到了。有时候我们不需要轰然的壮观，一个小小的片刻就能让我咧着嘴傻笑。人生往往那些轰然最为显赫，但是轰然远非全部。

在梦幻的蓝紫色天幕里飞行吧，哪怕只有一瞬。

它是某个夏日午后的回想，却让人久久难忘。这种放空与治愈在当下尤为可贵。

爬上山峰，只是想她说一句我爱你。

手机上读扫描版pdf：解决方案

发表于 2023-01-02 分类于 工具

1. 多看阅读

多看的重排功能对扫描版pdf还是卓有成效的，点击重排之后放大字体即可起到不错的效果。
但是只看本地书目的话多看的功能非常冗余，可以考虑安装旧版本

2. koreader

对于安卓设备来说，koreader似乎和其他应用有着不同，使用起来更像是在使用电纸书，没有即时反馈，翻页方式是点击翻页，重排效果还不错，但是在手机上使用起来实际体验感觉有点别扭…….如果是安卓系统的电纸书倒可以一试。
官网

3. k2pdfopt

电脑端的重排软件，导出重排后pdf，有很多参数可以调节，调教方法见此，全英但很好理解，支持win、mac及linux，手机的屏幕参数有在线测试的网站，搜索测试ppi似乎可以找到。
官网

不过说到底，复杂图多的pdf怎样重排效果都是不会太好的，教材等资料还是打印出来比较方便啊。

写给2022

发表于 2023-01-02 更新于 2023-01-04 分类于 随笔

上半年，一直在备战高考。要说不焦虑当然不太可能，但是似乎我已经找到了生活的模式，不再那么慌张和困惑了。只不过数学成绩一直不如人意，究竟到了最后也并不那么令人满意，但也只能如此了。说一点也没有遗憾是假的，在最初时每个人都似乎满怀雄心壮志，然而在最后咧嘴笑的却不是大多人。人生不就是这样，有期冀就会有遗憾，即使我相信我自己存在的意义，没能达到目标这一点似乎也足够令人沮丧。不过也就这样了，人总得往前看不是吗。

然后就是毕业。

毕业和高考的隆重与辉煌相比似乎仓促苍白多了，考完最后一科的那晚就必须要离开，没时间说一些煽情的话，很快的，我们就分开了。晚上我从宿舍拖了行李离开，连廊上的灯都亮起来了，超越楼的探照灯亮着，白晃晃的。我想起我们军训的时候也是那白晃晃的灯在晚上训练时照着，把我们的影子拉了很长。

曾无数次走过连廊，在教学楼后面的马路上聊天散步，在树林的亭子里聊天马行空。我们看着天上的云飘过去，一直飘到回忆里，才恍然大悟，那已经只是回忆了。

他在走之前说，要创造一些回忆，我其实在暗笑他傻，现在只是黯然的感伤。在无数个过去的日夜，我究竟得到了什么，又失去了什么，谁知道？我曾经那么努力地向闪闪发光的梦走去，可没能到达。我有那么多的希冀祈求与遗憾发生在这里，可现在就要告别了。

一月初的时候，我的姥姥不在了。

我哭了，大人们都哭了，我抱着的是她的骨灰盒，旁边放了她的新华字典，要让她在那边也能看到。我们把纸钱塞进炉子，它们从边上开始烧起来了，上面印着大大小小的图案，还有纸叠的元宝，金光闪闪的，火焰熊熊，用铁钎子向里捅，纸灰便飞出来。

在殡仪馆的门口，妈妈拿着那张探望的卡片，带着哭腔对我说，你姥姥现在只有一张卡片了。她把卡片揣进那个放满姥姥证件的卡包。

我说不出话来，只能紧紧抱住她。

死亡究竟是很简单的东西。轻盈的跨过那个门槛，她便不见了。

我想起很多，但我说不出话来。

2022充满了告别。和姥姥告别，和高中告别，和疫情防控告别。直至现在我也不知道怎么面对告别.

宝玉只愿常聚，而黛玉说，聚时欢喜，到散时岂不清冷。

整个暑假，充斥了惆怅，似乎也并没有多少对未来的期许。

八月大学开了学。

我见了我的室友，认领了宿舍，看了学校的各个角落。

然后是军训，我在四连，教官人很好（我私以为长得挺像虎哥），但是后来拍照时生病了没来。我也想过要不要问问他联系方式，但是还是作罢。留了又能怎么样呢。

军训前期很累，后来开始下雨，没法练行列式，其实还挺摆烂的。

值得一提的可能是过了合唱团面试，成为了男高的一员。

然后便开始上课，没什么特别的事情。上课，排练，写作文。没有特别要好的朋友，也没有值得感动的事情。一切都让我没什么感觉，钝感、麻木。我的感觉就是，没什么感觉。（青谷是不是有一首歌就叫，我没有活着的感觉）（sm33484862）

我时常怀疑自己出问题了。

似乎过于依赖互联网（主播女孩重度依赖？），我看了很多形形色色，但却仍然没什么感觉。（不过合唱的时候基本还是挺让人高兴的）

我仍然希望被理解和深度的交流，但这似乎太渺茫了。

后来便是疫情，开始上网课。

后来返乡了。

我仍然是这样，但似乎一切有些许不同。

其实发生了很多很宏大的事情，众说纷纭着

我不知道。

但我仍希望还有明天。

记梦

发表于 2023-01-01 更新于 2023-01-02 分类于 随笔

记梦5

他们说那叫“帝王冢”。

北京扩张的第十四年后，m城终于吃到了苦头。

或许用“终于”并不合适，毕竟苦头一直含在嘴里，只是现在愈发疼痛。

走出火车站，黄沙漫天，土地干裂，树木枯死，行人匆匆而过，无不带着厚厚的口罩……这是我印象中的m城吗？

为了保护环境，博物馆旁的少年宫在前几年被拆除了，种上了一片树林。当时的市长还在树林前微笑合影，新闻标题是《m城助力京津环境保护与经济发展》，现在树木日益枯，黑色枯木与黄色沙尘相映，如同某个一语成谶的梦，陡生黑色幽默之感。

而现在距当时的市长因贪污下台，也已经过去五年了。

曾经中华大街两侧的法桐被悉数砍去，站在博物馆广场前，那种熟悉又陌生的感觉令我苦笑。黄沙中的市博物馆像个饱经风霜的老人，苦哈哈地笑着，颤颤巍巍地站在那里，不知何时会倒下去。

他们说那叫“帝王冢”，是在都城周围零散分布的奇异景象，北京吸干抹净了整个h省后，被其触须紧紧包裹的m城首当其冲，也纳入了“帝王冢”的范畴。

傍晚下起了零零碎碎的冰雹，天总是昏暗而阴沉，令人憋闷，或许我记忆中的m城早已死了，埋在了帝王身侧，而现在的一切，不过是荒唐不羁的梦，我也终于会从梦中离去。

可我睁开眼的此时，我也难以分辨这究竟是不是梦了。

或许我的回忆才是假的吧。
